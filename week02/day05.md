# Day 05 - Choosing the Right AI for the Right Task

## Insight
Different AI tools behave differently even when given the same prompt.

For this API review task, ChatGPT was easy to understand, Claude provided strong explanatory context, and Codex gave the most implementation-oriented response.
This experiment showed me that choosing the right AI depends on the task, not the tool itself.

## Insight
There is no single "best" AI tool.

Each AI has different strengths, and the key is choosing the right one for the right task.
Rather than mastering one AI, I want to build a workflow that combines multiple AI tools effectively.
